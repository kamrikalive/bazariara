# Blueprint: Интернет-магазин товаров для сада и отдыха

## Обзор

Этот проект представляет собой простой интернет-магазин, созданный с использованием Next.js и Firebase. Приложение позволяет пользователям просматривать товары из двух категорий - "Сад" и "Товары для отдыха", добавлять их в корзину и оформлять заказ, предоставляя свои контактные данные.

## Дизайн и стили

- **Основной макет:** Мы использовали корневой `layout.tsx` для создания единой структуры с шапкой (Header) и основным контейнером для контента.
- **Стили:** Для стилизации мы использовали Tailwind CSS. Цветовая схема основана на оттенках серого, с акцентным зеленым цветом (lime-500) для кнопок и интерактивных элементов.
- **Компоненты:**
    - `Header`: Содержит логотип, навигационное меню, иконку корзины и аватар пользователя.
    - `Card`: Отображает информацию о товаре (изображение, название, цена) и кнопку добавления в корзину.
    - `CardList`: Отображает список карточек товаров.

## Реализованные функции

- **Просмотр товаров:** Пользователи могут просматривать товары на главной странице.
- **Фильтрация по категориям:** Товары можно фильтровать по категориям.
- **Пагинация:** На главной странице отображается не более 20 товаров, с возможностью перехода по страницам.
- **Страница продукта:** Реализована отдельная страница для каждого продукта.
- **Корзина:**
    - Пользователи могут добавлять товары в корзину с главной страницы или со страницы продукта.
    - Содержимое корзины сохраняется в `localStorage`.
    - Создана страница корзины, где можно изменять количество товаров или удалять их.
    - В шапке сайта отображается иконка корзины с количеством товаров.
