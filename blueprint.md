# Blueprint: Интернет-магазин товаров для сада и отдыха

## Обзор

Этот проект представляет собой простой интернет-магазин, созданный с использованием Next.js и Firebase. Приложение позволяет пользователям просматривать товары из двух категорий - "Сад" и "Товары для отдыха", добавлять их в корзину и оформлять заказ, предоставляя свои контактные данные. При оформлении заказа менеджер получает уведомление в Telegram.

## Дизайн и стили

- **Основной макет:** Мы использовали корневой `layout.tsx` для создания единой структуры с шапкой (Header) и основным контейнером для контента.
- **Стили:** Для стилизации мы использовали Tailwind CSS. Цветовая схема основана на оттенках серого, с акцентным зеленым цветом (lime-500) для кнопок и интерактивных элементов.
- **Компоненты:**
    - `Header`: Содержит логотип, навигационное меню, иконку корзины, **общую сумму заказа** и аватар пользователя.
    - `Card`: Отображает информацию о товаре (изображение, название, цена) и кнопку добавления в корзину.
    - `CardList`: Отображает список карточек товаров.

## Реализованные функции

- **Просмотр товаров:** Пользователи могут просматривать товары на главной странице, загружая их из базы данных Firestore.
- **Фильтрация по категориям:** Товары можно фильтровать по категориям, которые динамически создаются на основе данных из Firestore.
- **Пагинация:** На главной странице отображается не более 20 товаров, с возможностью перехода по страницам.
- **Страница продукта:** Реализована отдельная страница для каждого продукта с загрузкой данных из Firestore.
- **Корзина:**
    - Пользователи могут добавлять товары в корзину.
    - Содержимое корзины сохраняется в `localStorage`.
    - Создана страница корзины, где можно изменять количество товаров или удалять их.
    - В шапке сайта отображается иконка корзины с количеством товаров.
    - В шапке сайта, слева от иконки корзины, теперь отображается общая сумма заказа.
    - **Новое:** На странице корзины для управления количеством товаров добавлены кнопки "+" и "-", что улучшает пользовательский опыт на мобильных устройствах.

## План текущих изменений

1.  **Создать страницу оформления заказа (`/checkout`):**
    - На странице будет отображаться итоговый список товаров из корзины.
    - Будет добавлена форма для ввода имени и контактных данных (номер телефона или ник в Telegram).

2.  **Настроить отправку уведомлений в Telegram:**
    - Создать серверное действие (Server Action), которое будет обрабатывать отправку формы.
    - Это действие будет форматировать данные заказа и отправлять их в виде сообщения в чат Telegram с помощью бота.

3.  **Создать страницу подтверждения заказа (`/order-success`):**
    - После успешного оформления заказа пользователь будет перенаправлен на эту страницу.
